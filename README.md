# Breaking Bad EDA

## 📌 Project Overview
This project focuses on Exploratory Data Analysis (EDA) of the Breaking Bad dataset. The objective is to analyze episode ratings, viewer engagement, and season-wise trends to gain meaningful insights from the data. Various data cleaning, visualization, and statistical analysis techniques have been applied using Python.

## 📊 Dataset Information
The dataset contains information related to Breaking Bad episodes, including season numbers, episode numbers, IMDB ratings, episode duration, and viewer statistics. The dataset was analyzed to understand patterns, trends, and relationships among different variables.

## 🛠️ Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🔍 Data Cleaning Process
- Checked dataset structure and data types
- Identified and handled missing values
- Removed duplicate records (if any)
- Generated descriptive statistics for numerical features

## 📈 Exploratory Data Analysis
The following analyses were performed:
- Distribution of IMDB Ratings
- Episode Rating Trends
- Season-wise Analysis
- Viewer Distribution Analysis
- Correlation Analysis
- Pairplot Visualization
- Heatmap Visualization
- Comparison of Ratings Across Seasons
- Relationship Between Viewers and Ratings

## 📊 Key Insights
- Episode ratings vary significantly across seasons.
- Certain episodes achieved exceptionally high IMDB ratings.
- Viewer engagement shows noticeable fluctuations throughout the series.
- Some seasons maintain consistently strong ratings.
- Correlation analysis helps identify relationships between numerical variables.
